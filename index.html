Py-Hurts-My-Arm

A Lightweight Python Code Editor

About the App

Py-Hurts-My-Arm is a free, lightweight Python code editor made in Python. It is designed for users who want a fast, simple editor without the complexity of larger IDEs like PyCharm.

The name "Py-Hurts-My-Arm" comes from the fact that I don't like PyCharm, and wanted a more efficient, lightweight tool for coding.

Features

  • Syntax Highlighting for Python code
  • Multi Themes + Block Cursor
  • Key Binds in the View Tab
  • Edit Tab: Copy, Cut, Paste, Undo, Redo
  • File Tab for managing files
  • Run Button to execute code directly within the editor

Installation & Running

1. Download the compiled release from the Releases page.

2. Extract the files if necessary.

3. Run the executable: pyhurtsmyarm.exe.

Limitations

  • No multi-file editing
  • No auto-suggestions or auto-indent
  • No error detection
